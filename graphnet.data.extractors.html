class graphnet.data.extractors.i3genericextractor.I3GenericExtractor(keys=None, exclude_keys=None)[source]

Bases: I3Extractor

Dynamically and generically extract information from frames.

This class parses all keys in the I3Frame objects it is called on, and tries to automatically cast all of the available information to pure-python classes. This is done recursively, for each object in the I3Frame, by looking for member variables that can be parsed; by looking for objects that have signatures similar to python lists or dicts; and by handling a handful of special cases: - Pulse series maps, - Per-pulse maps, - MC tree, and - Triggers.

Parameters

  • keys (Union[str, List[str], None], default: None) –

  • exclude_keys (Union[str, List[str], None], default: None) –

__call__(frame)[source]

Extract all possible data from frame.

The following types of objects are handled as special cases: - Pulse series maps, - Per-pulse maps, - MC tree, and - Triggers.

All other fields are cast to pure-python classes by generically parsing member variables, and checking if the object has a signature similar to python lists or dicts.

Return type

Dict[str, Any]

Returns

Dictionary containing each parsed key in frame, and the

corresponding, extracted data in pure-python format.

_muon_stopped(truth, borders, shrink_horizontally=100.0, shrink_vertically=100.0)[source]

Calculate whether a simulated muon within the detector volume.

IMPORTANT: The final position of the muon is saved in truth extractor/ databases as position_x, position_y and position_z. This is analogouos to the neutrinos whose interaction vertex is saved under the same name.

Parameters

  • truth (Dict[str, Any]) – Dictionary of already extracted truth-level information.

  • borders (List[ndarray]) – The first entry are the (x,y) coordinates, the second entry is the z-axis min/max depths. See I3TruthExtractor constructor for hard-code example.

  • shrink_horizontally (float, default: 100.0) – Shrink (x,y)-plane further with exclusion zone. Defaults to 100 meters. shrink_vertically: Further shrink detector depth with exclusion height. Defaults to 100 meters.

  • shrink_vertically (float, default: 100.0) –

Return type

Dict[str, Any]

Returns

Dictionary containing the (x,y,z)-coordinates of final the muon

position as well as a boolean indicating whether the muon stopped within the chosen fiducial volume.
